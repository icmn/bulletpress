EPR Checker

A bullet analysis tool

Instructions

This tool looks at bullets in the textbox on the left and provides metadata and analysis of them in the tabs on the right.

To get started, add some bullets to the textbox on the left. The information on the right will be automatically updated as you edit or add to the bullets.


There are currently four types of information that are provided:

Abbreviation conflicts: this is a list of words whose letters can all be found in other words. Bold pairs of words include one or more words not normally found in a dictionary, increasing the likelihood that an abbreviation was used.

Acronyms: this is a list of all fully capitalized words found in the bullets. If you type a description into an acronym's text box, this will cause that acronym and its description to appear in the generated remarks section found below the acronym list.

Word counts: this is a list of all words and their number of occurrences in the text. This can sometimes help call out word overuse, i.e. the bullets using the same words start sounding the same.

Numbers: this is a list of all of numbers found in the bullets. This is mostly just for informational purposes.
